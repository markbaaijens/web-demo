# web-demo
Demo for web-app; 3-tier; rest-api written in flask/python

## Requirements
OS:
- python3
- sqlite3 (apt install sqlite3)

Python packages
- pip install -r requirements.txt

## Run application
Start API
- cd api
- python controller.py
Access: localhost:5000

Start webserver:
- cd web/flask
- python controller.py
Access (browser): localhost:5001
